# openwrt-mentohust-apk
mentohust锐捷验证插件，ipk版本在25版本以上的openwrt不再适用，于是编译出apk版（包含二进制程序与 LuCI 界面）。


---

```text
📌 项目背景
======================================================================
由于 mentohust 锐捷验证插件的传统 .ipk 版本在 25版本及以上 的新版 OpenWrt
系统中不再适用，导致许多校园网同学升级系统后无法进行网络认证。

为了解决这一痛点，本项目特此编译了 .apk 格式 的版本，完美适配新版 OpenWrt
系统的软件包管理架构，帮助大家继续愉快地畅享校园网！


✨ 功能特点
======================================================================
* 双剑合璧：同时提供底层内核程序与直观的 LuCI 网页后台控制面板。
* 完美兼容：专门针对 OpenWrt 25+ 版本的 APK 包管理器进行优化。
* 稳定认证：继承 MentoHUST 优秀的锐捷认证机制，断线自动重连。


🚀 下载与安装指南
======================================================================
1. 下载插件：
   前往本仓库右侧的 Releases (发行作品) 页面，下载以下两个 .apk 文件：
   * mentohust-0.3.1-r2.apk （核心验证程序）
   * luci-app-mentohust-27.173.apk （网页管理界面）

2. 命令行一键安装（推荐 💻）：
   使用 SSH 工具连接到 OpenWrt 路由器，将文件传至 /tmp 目录，然后依次运行：

   apk add --allow-untrusted /tmp/mentohust-0.3.1-r2.apk
   apk add --allow-untrusted /tmp/luci-app-mentohust-27.173.apk

   ⚠️ 注意：由于是个人编译版本，必须加上 --allow-untrusted 参数。

3. 网页端安装方式（备选 🌐）：
   a. 登录你的 OpenWrt 后台（LuCI 管理界面）。
   b. 导航至：系统 -> 软件包 (System -> Software)。
   c. 点击 "上传软件包"，先上传并安装 mentohust-0.3.1-r2.apk。
   d. 随后再次点击上传，安装 luci-app-mentohust-27.173.apk。


⚙️ 开始配置
======================================================================
1. 安装完成后，刷新 OpenWrt 网页后台。
2. 在菜单栏中找到 MentoHUST 或 锐捷认证 插件。
3. 正确填写你的：校园网账号、密码以及对应的认证网卡（如 eth1）。
4. 点击 "保存并应用" 即可开始认证。
